### changeItemsPerPage

Changes items per page

```jsx
const [
  ...
  {
    changeItemsPerPage
  }
] = useFetchPagination(...)
```

### changePage

Changes current page

```jsx
const [
  ...
  {
    changePage
  }
] = useFetchPagination(...)
```

### changePagination

Changes both current page and items per page

```jsx
const [
  ...
  {
	changePagination
  }
] = useFetchPagination(...)
```

### getPaginationHref

Builds href for use on navigation. (usually used with pagination component)

```jsx
const [
  ...
  {
    getPaginationHref
  }
] = useFetchPagination(...)
```
